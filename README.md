# CC-PM: Claude Code 專案管理助手

一個基於 Claude Code 的輕量級專案管理工具，透過對話方式管理專案進度、追蹤任務、記錄決策。

## 快速開始

1. 在專案目錄下啟動 Claude Code
2. 執行 `/start` 開始初始化專案
3. 回答問題後，系統會自動建立專案結構

## 檔案結構

```
.
├── CLAUDE.md        # AI 助手指令（請勿修改）
├── project.yaml     # 專案基本資訊
├── tasks.md         # 任務清單
├── timeline.md      # 時程與里程碑
├── decisions.md     # 決策記錄
├── meetings/        # 會議記錄
├── docs/            # 專案文件、產出
└── assets/          # 參考資料、圖片
```

## 常用操作

| 你說 | 助手會做 |
|------|----------|
| `/start` 或「開新專案」 | 啟動專案初始化流程 |
| 「新增任務：XXX」 | 新增任務到 tasks.md |
| 「更新進度」或「XXX 完成了」 | 更新任務狀態 |
| 「目前狀態」 | 顯示專案進度摘要 |
| 「記錄決策：XXX」 | 記錄到 decisions.md |
| 「開會記錄」 | 建立會議記錄 |

## 任務狀態

| 狀態 | 說明 |
|------|------|
| `backlog` | 待辦 - 已規劃但未開始 |
| `in-progress` | 進行中 |
| `blocked` | 阻塞 - 需要處理 |
| `review` | 審核中 |
| `done` | 已完成 |

## 優先級

| 等級 | 說明 |
|------|------|
| P0 | 緊急且重要，必須優先處理 |
| P1 | 重要，本週需完成 |
| P2 | 一般，時間允許再做 |
